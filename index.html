Sistem Pemasaran Toko Music

Aplikasi pemesanan alat musik secara online yang dibangun menggunakan React untuk frontend dan Node.js untuk backend. Sistem ini memungkinkan pengguna untuk melihat, memesan, dan membayar berbagai alat musik dan aksesoris. Data disimpan di MongoDB untuk kemudahan pengelolaan inventaris.

Peran: Full Stack Developer
Tech Stack: React, Node.js, MongoDB, Express
Pencapaian: Mengimplementasikan sistem pemesanan yang responsif, mengurangi kesalahan manual dalam pemesanan hingga 20%.
